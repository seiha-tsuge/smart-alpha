# smart-alpha

## 1. プロダクト要求

### 1.1 背景

株価の変動要因の中で最も影響のある要因は、企業業績であると言われている。投資家は、企業業績に影響を及ぼす多様な情報を取捨選択し、企業業績の分析・予測している。しかし、多様な情報の全てを、個人投資家が自分で目を通して企業分析に用いることは不可能に近い。そのため、情報技術を活用し、企業分析に適用するのが本プロジェクトの目的である。

### 1.2 システム概要

上記背景に鑑み、個人投資家の企業分析を支援するシステムを構築し、企業分析にかかるコストを低減することが重要である。企業分析支援システム（以下、SmartAlpha）は、決算速報のデータを基にデータを分析することで、個人投資家が分析に注力すべき企業を絞り込む。具体的には、決算短信の情報に加え、Quality/Value/Momentumの情報を提示する。将来的には、人工知能を活用し、個人投資家が企業分析にかかるコストをゼロにする。さらに、人工知能が予測した未来の決算短信を個人投資家に提供することで、近い将来に発生したであろう企業分析のコストも無くすシステムとする。また、SmartAlphaは個人投資家が企業分析について意見の交換や議論を行う場を提供するために、コミュニティを形成する機能を提供する。
