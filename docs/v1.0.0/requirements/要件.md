### 非機能要求
- エラー検知ツールを導入すること

### 機能

---

**基本機能**

- ユーザーが財務諸表を分析できること

    以下の内容を見れること

    - 企業コード
    - 企業
    - 発表時間
    - いつの四半期決算
    - 増益などのステータス
        - 増益・減益・赤字拡大・赤字転落・赤字縮小・黒字転換・増収増益・増収減益・減収増益・減収減益
    - 売上高・営業利益・経常利益・税引前当期純利益・当期純利益
        - 上記の前年同期比
    - 業績予想
        - 進捗率
        - 前年の進捗率
    - 期初業績予想
        - 進捗率
        - 前年の進捗率
    - 受注残

- 指定日付に財務諸表を発表した企業一覧
    - 決算短信あるいは有価証券報告書か
    - 上方修正あるいは下方修正か

- 全企業の最新財務諸表を見れること

**投資候補**
- 投資候補の企業を登録・削除できること
- 投資候補の企業一覧を見れること
- 他の人が登録した投資候補を見れること

**検索系**

- 指定した日付に発表された財務諸表を検索できること
- 条件に合った企業のみ表示できること
    - 増益などのステータス
    - 表示項目の前年同期比がマイナスの企業を表示しない
    - 表示項目の前年同期比がある数値以上または以下

**オプション**
- 行全体をバフェットコードの詳細ページへのリンクにすること
